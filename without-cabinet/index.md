# Without Cabinet

This side of the demo is meant to feel messy on purpose.

The work is not bad. The system is.

Outputs exist, but they are disconnected:

- one session writes research
- another session writes content
- a third session forgets what already shipped
- extra files get saved into ad hoc folders with no shared memory layer

## What This Side Proves

- good outputs are not enough if the surrounding system resets every time
- files drift into separate folders with no clear source of truth
- structured tracking exists, but it is stale and disconnected
- every new session depends on a human reconstructing the past

The result is not "bad AI."

It is expensive memory work pushed back onto the user.
