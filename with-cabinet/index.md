# With Cabinet

This side of the demo is meant to feel like an operating system, not a pile of outputs.

Everything useful lives in one place:

- recurring agents
- persistent knowledge
- structured sheets
- reusable deliverables
- visual context
- lightweight apps that run directly from files on disk

## What This Side Proves

- context survives across runs
- files are connected to each other instead of scattered
- agents read the same source of truth before doing work
- structured data can live next to narrative docs
- dashboards and lightweight tools can live inside the same workspace

The important shift is not just "the AI remembers."

It is that the whole workspace becomes legible, navigable, and reusable.
