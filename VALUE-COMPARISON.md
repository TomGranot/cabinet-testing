## Same goal. Same AI. Completely different outcome.

Both scenarios use Claude. Both produce good content. But only one gets better over time.

| Session 3 | 12 min re-explaining | 0 min |
| Session 4 | 15+ min | 0 min |

> Without Cabinet, you spend more time managing context than producing output.

## What You Actually Get

| Cost | Claude subscription | Cabinet is free + open source |

---

## The Core Insight

Without Cabinet, the AI is powerful but amnesiac.
You become the memory layer — the glue between sessions.
That's not leverage. That's a second job.

With Cabinet, the files ARE the memory.
Agents read them, update them, react to them.
You don't manage the system. The system manages itself.

**The AI doesn't get smarter. Your setup does.**
